Blackjack

This was the first webapp I built for Tealeaf Academy. It is a simple Black Jack game built using Sinatra. It was a great refresher on OOP and HTTP requests and routing. I added a scoreboard functionality to mine to help the user keep track of how much they had, what they started with and how much they were up or down.

Testing Out the Application

Just visit the site and follow the instructions, no login necessary.

 

Skills

  • Bootstrap, CSS, Ajax
  • Sinatra
  • Deployment to heroku
  • HTTP requests

 

Webapp Basics

This application was a great starting point for my classes. Before diving into Ruby on Rails, I got to learn about some of the “magic” behind it by building an app from scratch using Sinatra. Going through all the steps required to get it working allowed me to get an understanding of just what processes Rails wraps away and takes care of for you.
